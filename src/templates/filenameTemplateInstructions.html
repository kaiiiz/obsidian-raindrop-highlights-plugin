Filename template (Nunjucks) for creating synced Raindrop.io highlights & annotations.

Some notes:

  • The rendered result is used as the filename for the bookmark.
  • The rendered result is sanitized and truncated to 255 bytes.
  • The plugin will reject the invalid template.
  • If the file already exists in the vault, the auto generated suffix will be appended to the rendered result and used as the filename.
  • Available variables to use are the same as the previous template.
