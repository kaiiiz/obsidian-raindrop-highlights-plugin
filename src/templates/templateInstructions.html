Template (Nunjucks) for rendering every synced Raindrop.io highlights & annotations.

Available variables to use

Metadata
  • {{now}} (Moment) - Present time
Article Metadata
  • {{is_new_article}} (bool) - New file indicator
  • {{id}} (number) - Article identifier
  • {{title}} (string) - Title
  • {{excerpt}} (string) - Article excerpt
  • {{note}} (string) - Article note
  • {{link}} (string) - Link to source
  • {{highlights}} (Highlight[]) - List of your Highlights (Detail attributes refer to the following Highlight section)
  • {{collection}} (Collection) - Collection data (Detail attributes refer to the following Collection section
  • {{creator}} (Creator) - Creator data (Detail attributes refer to the following Creator section
  • {{tags}} (string[]) - List of tag
  • {{cover}} (string) - Article cover
  • {{created}} (Moment) - Created on
  • {{type}} (string) - Article type
  • {{important}} (bool) - Favorite article
  • {{raindropUrl}} (string) - Link to raindrop page
Creator
  • {{name}} (string) - Creator name
  • {{id}} (number) - Creator id
Collection
  • {{title}} (string) - Collection title
Highlight
  • {{id}} (string) - Highlight identifier
  • {{text}} (string) - Text
  • {{color}} (string) - Highlight color
  • {{created}} (Moment) - Created on
  • {{lastUpdate}} (Moment) - Updated on
  • {{note}} (string) - Annotation
Filter
  • date(format: string) Convert Moment object to string - example: {{ now | date("YYYY/MM/DD") }}. For more syntax, refer to format reference.
